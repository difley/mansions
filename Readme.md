----------------------
Mansions
----------------------

Use qb64 to compile mansions.bas

    ./qb64 -c ./mansions.bas


Run the application with

    ./mansions


----------------------
Map Editor and Viewer
----------------------

Use qb64 to compile mapmaker.bas

    ./qb64 -c ./mapmaker.bas


Run the application with

    ./mapmaker
